# TPredictX7

Custom DDNet client forked from Tater/DDNet, with a focus on prediction systems, utility, and competitive quality-of-life features.

## Highlights

- Advanced prediction systems for local and other players.
- Gameplay tools including Fast/Gamma Input, Snap Tap, Hook Combo, Speedrun Timer, and Finish Prediction.
- Utility features for enhanced communication and client-side indicators.

## Installation

- Clone the repository and build it using the standard DDNet build flow.
- Or open the project in your existing DDNet toolchain and compile the client binaries from source.
